
# Employee Management System – Backend

## About the Project

This is a simple **Node.js and Express** backend project created for the **Day 3 KT task**.
It performs basic employee management operations using REST APIs.

---

## Technologies Used

* Node.js
* Express.js
* JavaScript
* Postman

---

## How to Run

1. Clone the repository

```bash
git clone https://github.com/midhun3053/day3-employee-management.git
```

2. Go to project folder

```bash
cd day3-employee-management
```

3. Install dependencies

```bash
npm install
```

4. Start server

```bash
node server.js
```

Server runs on:

```
http://localhost:3000
```

---

## API Endpoints

### Get all employees

```
GET /employees
```

### Add employee(s)

```
POST /employees
```

### Update one employee

```
PUT /employees/:id
```

### Update multiple employees

```
PUT /employees
```
